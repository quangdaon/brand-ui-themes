# Quangdao's Brand UI Theme

Theme configurations for my personal brand on various editors.

Built from my core palette:

| Hex       | Role                                 |
| --------- | ------------------------------------ |
| `#FFA630` | Amber — types, accents, warnings     |
| `#9F2042` | Crimson — keywords, tags, properties |
| `#222E50` | Navy — surfaces, borders, structure  |
| `#224102` | Forest — strings, constants, success |
| `#04080F` | Near-black — editor backgrounds      |
| `#F7F7F9` | Off-white — primary text             |

## Editors

| Folder                    | Editor                                              |
| ------------------------- | --------------------------------------------------- |
| [`zed/`](./zed)           | [Zed](https://zed.dev)                              |
| [`vscode/`](./vscode)     | [Visual Studio Code](https://code.visualstudio.com) |
| [`obsidian/`](./obsidian) | [Obsidian](https://obsidian.md)                     |

Each folder contains its own setup instructions.

## Disclaimer

These themes were generated with the help of AI (Claude) based on the brand colors from [quangdao.com](https://quangdao.com). The core palette is original; the derived shades, UI mappings, and syntax highlighting assignments were produced through AI-assisted iteration.
